## Requirements 

Python 3

## How Install It?

```
$ git clone https://github.com/vernieri/Apoteose.git

```
## How I Run it?

1. after clone my repo open terminal
2. go to the Apoteose folder
3. Type:
```
$ python example.py
``` 
4. Now, just import Apoteose and use on your own Projects using:
5. from apoteose import *
